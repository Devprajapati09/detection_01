# Smart Agriculture System

This repository contains the codebase for a comprehensive smart agriculture platform, including frontend, backend, AI services, Supabase integration, and documentation.

## Project Structure

- `frontend/` – React/Vue frontend application.
- `backend/` – Node/Express API server.
- `ai-services/` – AI models and services.
- `supabase/` – Supabase configuration and database.
- `docs/` – Project documentation.

## Getting Started

1. Clone the repository.
2. Install dependencies for each subproject.
3. Configure environment variables in `.env`.
4. Run services via Docker Compose.
